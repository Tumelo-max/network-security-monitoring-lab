# Python Network Security Monitoring Lab

## Overview

This project demonstrates network traffic monitoring and basic threat detection using Python.

The lab simulates how Security Operations Center (SOC) monitoring systems analyze network traffic and detect suspicious activity.

## Features

- Packet sniffing
- DNS monitoring
- Intrusion detection
- Port scan detection
- Log analysis
- Real-time alert system

## Labs Included

Lab 1 – ICMP Packet Sniffer  
Lab 2 – HTTP Traffic Sniffer  
Lab 3 – Packet Logger  
Lab 4 – DNS Sniffer  
Lab 5 – DNS Threat Detector  
Lab 6 – Intrusion Detection System  
Lab 7 – Port Scan Detector  
Lab 8 – Log Analyzer  
Lab 9 – Security Alert System

## Technologies Used

- Python
- Scapy
- Linux
- Network packet analysis

## Purpose

This project was built as part of a cybersecurity learning lab to understand how network monitoring and threat detection work in real environments.
